## アイデア1: 「冒険地図帳」スタイル — ヴィンテージ探検マップ

### Design Movement
19世紀の探検地図・航海図をモチーフにしたヴィンテージカートグラフィースタイル。古い羊皮紙の質感と手描き風のイラストを組み合わせた、宝の地図のようなビジュアル。

### Core Principles
1. **手描き感**: 全てのUI要素に手描き風のタッチを加え、温かみと冒険感を演出
2. **羊皮紙テクスチャ**: 背景やカードに古い紙の質感を使い、歴史的な雰囲気を醸成
3. **コンパスローズ**: 方位磁針モチーフを装飾的に配置し、探検テーマを強調
4. **木目フレーム**: UIパネルに木目調のフレームを使い、船のキャビンや冒険者のデスクを連想させる

### Color Philosophy
- 背景: 羊皮紙色（#F5E6C8）— 古い地図の温かみ
- プライマリ: 深い茶色（#5C3D2E）— 木とインクの色
- アクセント: 航海の赤（#C0392B）— 重要なアクション
- 資源色: ゴム=深緑、石油=墨色、金=アンティークゴールド、食料=小麦色
- 海: 手描き風の波模様を持つ青緑（#2E86AB）

### Layout Paradigm
画面全体を「広げた地図」として構成。上部に相手プレイヤー情報を羊皮紙の巻物風に、中央に六角形マップを地図風に、下部に自分の情報を木製デスク風パネルに配置。

### Signature Elements
1. 羊皮紙の巻物風カードとダイアログ
2. コンパスローズ型のターンインジケーター
3. 手描き風の波と船のアニメーション

### Interaction Philosophy
タップすると「インクが滲む」ようなリップルエフェクト。カードは巻物を広げるように展開。

### Animation
- サイコロ: 木製サイコロが転がるアニメーション
- 資源獲得: 宝箱が開くモーション
- イベント: 巻物がスルスルと開く演出
- ターン切替: コンパスの針が回転

### Typography System
- 見出し: "Pirata One" — 冒険的な雰囲気
- 本文: "Noto Sans JP" — 読みやすさ重視
- 数字: "Cinzel" — 古典的な数字表現

## アイデア2: 「ポップ冒険RPG」スタイル — アニメ風ゲームUI

### Design Movement
日本のモバイルゲーム・アニメRPGのUIデザインを参考にした、カラフルでポップなスタイル。参考画像に最も近い方向性で、キャラクター性とゲーム感を重視。

### Core Principles
1. **ゲーム感**: ステータスバー、アイコンバッジ、光るエフェクトなど、ゲームUIの文法を活用
2. **カラフル＆明快**: 各要素が色で明確に区別され、一目で意味がわかる
3. **レイヤー構造**: 背景→マップ→UI→ポップアップの明確な奥行き
4. **タッチファースト**: 大きなボタン、明確なタップ領域、指で操作しやすいサイズ

### Color Philosophy
- 背景: 海の青グラデーション（#1B4F72→#85C1E9）— 冒険の舞台
- UIフレーム: 木目調の茶色（#8B6914）— ボードゲーム感
- プライマリ: 鮮やかな緑（#27AE60）— ポジティブアクション
- 警告: 暖かい赤（#E74C3C）— 注意喚起
- 資源色: ゴム=鮮緑(#2ECC71)、石油=ダークグレー(#2C3E50)、金=ゴールド(#F1C40F)、食料=オレンジ(#E67E22)
- カード背景: クリーム(#FDF2E9)

### Layout Paradigm
モバイルゲーム風の縦型レイアウト。上部バーに対戦相手情報、中央にヘックスマップ（ピンチズーム対応）、下部に自分のステータスと行動ボタンを配置。ポップアップは画面中央にオーバーレイ表示。

### Signature Elements
1. 木目フレーム付きのステータスパネル（画像のような装飾枠）
2. キラキラ光るエフェクト付きの資源アイコン
3. リボン型のラベルとバッジ

### Interaction Philosophy
ボタンは押すと「ぷにっ」と凹むスプリングアニメーション。重要なアクションには確認ダイアログではなく、ボタン長押しで実行。

### Animation
- サイコロ: 3D風に回転して結果表示、数字が弾むように出現
- 資源獲得: アイコンが飛んできて手持ちに吸い込まれる
- イベント: カードが画面中央にフリップして登場
- 建設: タイル上に建物がポンと建つモーション
- ターン切替: 画面が軽くフラッシュして次プレイヤーの色に

### Typography System
- 見出し: "M PLUS Rounded 1c" Bold — 丸みがありつつゲーム感
- 本文: "Noto Sans JP" Medium — 読みやすさ
- 数字/スコア: "Fredoka One" — ポップで目立つ数字
- ラベル: "Noto Sans JP" Bold — 明確なラベリング
</text>

## アイデア3: 「ミニチュアジオラマ」スタイル — 立体的なクレイ風

### Design Movement
クレイアニメーション・ミニチュアジオラマの世界観。全てのUI要素が粘土や木で作られたような質感を持ち、温かみのある3D的な表現。

### Core Principles
1. **触れそうな質感**: ボタンやカードに粘土・木・布の質感を持たせる
2. **丸みと柔らかさ**: 角を丸くし、全体的に柔らかい印象
3. **影と奥行き**: ドロップシャドウを効果的に使い、ジオラマ的な立体感
4. **手作り感**: 完璧すぎない、少し歪んだ手作り感のあるデザイン

### Color Philosophy
- 背景: 柔らかいスカイブルー（#D4E6F1）— ジオラマの空
- 地面: テラコッタ（#CD6155）とグリーン（#82E0AA）
- UIパネル: クリーム色の粘土（#FAE5D3）
- プライマリ: 明るい赤（#E74C3C）— 粘土の赤
- 資源色: ゴム=ライムグリーン、石油=チャコール、金=レモンイエロー、食料=ペールオレンジ

### Layout Paradigm
画面を「テーブルの上のジオラマ」として構成。マップはテーブル中央に置かれたジオラマ、UIパネルはテーブルの端に置かれた小物として配置。

### Signature Elements
1. 粘土質感のボタンとアイコン
2. ミニチュアの建物と船のイラスト
3. 布テクスチャのカード背景

### Interaction Philosophy
タップすると粘土が「ぐにっ」と押される感触。ドラッグは粘土を引っ張るような動き。

### Animation
- サイコロ: 粘土のサイコロがコロコロ転がる
- 資源獲得: ミニチュアの箱から資源が飛び出す
- イベント: 粘土のカードがぺたんと置かれる
- 建設: 粘土の建物がにょきっと生える

### Typography System
- 見出し: "Kosugi Maru" — 丸くて可愛い
- 本文: "Noto Sans JP" — 読みやすさ
- 数字: "Bubblegum Sans" — ポップな数字
